# NOE Distance Restraints Datasets

This repository contains experimental Nuclear Overhauser Effect (NOE) distance restraint datasets for several peptide and protein systems.

## Files

### Cyclic Peptides
- **`noe_CYC2.csv`**  
  NOE distance restraints for the **cis-cyclic peptide**.

- **`noe_CYC2t.csv`**  
  NOE distance restraints for the **trans-cyclic peptide**.

### Mini-Protein
- **`trp_noe.csv`**  
  NOE distance restraints for the **Trp-cage mini-protein**.

### CLN025 Peptide
- **`cln_noe_unfiltered_unfolded.csv`**  
  NOE distance restraints for **CLN025**, used in simulations initiated from the **extended structure**.  
  - Intraresidue distances have been **filtered out**.

- **`cln_noe_filtered_folded.csv`**  
  NOE distance restraints for **CLN025**, used in simulations initiated from the **NMR structure**.  
  - Intraresidue distances have been **filtered out**  
  - Distances involving the **first and last two residues** have also been **removed**
