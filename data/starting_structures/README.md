# Data Descriptions

- **ALA**: Alanine dipeptide

- **CYC2**: Cis-cyclic peptide  
- **CYC2t**: Trans-cyclic peptide  

- **P13n**: Folded polyproline (PII state)  
- **P13e**: Misfolded polyproline (alpha-helix state)  

- **TRP**: Folded Trp-cage mini-protein  
- **TRPu**: Misfolded Trp-cage mini-protein  

- **CLN**: Folded CLN025  
- **CLNunf**: Extended (unfolded) CLN025  

- **CHICF**: Folded chignolin  
- **CHIC**: Extended chignolin
